# Notes for Software

## Virtualization
- Software that creates an abstraction layer over computer hardware.
    - Basically creates a new computer but with shared hardware. 

## VMware
- Virtualization software
    - Rich features and scalability
    - Licensing/Implementation costs 
        - No perpetual licenses
        - recent per core cost hike, ~50$ per core per year
        - basic VMware kit no longer available
        - Owned by Broadcom

## VirtualBox
- Virtualization software
    - comparable features to VMware but still under
    - lacks advanced manual snapshot setup
    - broader range of network modes
    - Free and opensource
    - Owned by Oracle

### Kali Linux
- cybersecurity os
    - terrible daily driver with bad habits
    - runs as root by default, need to add priviledge boundaries against malware etc.
    - bloated with tools, actually increases your own attack surface against you

### Parrot OS
- cybersecurity os
    - most direct alternative to kali
    - lighter than kali 
    - similar tools to kali

### BlackArch
- cybersecurity os
    - Arch Linux
    - large tool repository
    - better for granular control
    - NOT beginner friendly

### REMnux
- cybersecurity os
    - specifically good for malware analysis
    - purpose-built for reverse engineering/forensics
