## Interpreter
- Translating 
    - i.e. Python **interprets** the code into machine code *during* the program runs

## Python
- Interpreted language
- bit of a hybrid
- Does bytecode compilation step first
    - hence the .pyc files that show up
- BUT, it still needs to translate the bytecode to machine code, thus the interpreter designation
