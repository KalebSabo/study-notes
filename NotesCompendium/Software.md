# Notes for Software

## Virtualization
- Software that creates an abstraction layer over computer hardware.
    - Basically creates a new computer but with shared hardware. 

## VMware
- Virtualization software
    - Rich features and scalability
    - Licensing/Implementation costs 
        - No perpetual licenses
        - recent per core cost hike, ~50$ per core per year
        - basic VMware kit no longer available
        - Owned by Broadcom

## VirtualBox
- Virtualization software
    - comparable features to VMware but still under
    - lacks advanced manual snapshot setup
    - broader range of network modes
    - Free and opensource
    - Owned by Oracle
